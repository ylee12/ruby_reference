
To create a RVM enable directory, create the .ruby-gemset and .ruby-version files with the 
ruby version and gemset/project name.

ylee@ylee-hp-6200:~/pj/ruby_exercise/rspec_square$ vi .ruby-gemset
ylee@ylee-hp-6200:~/pj/ruby_exercise/rspec_square$ vi .ruby-version

Use Rspec

ylee@ylee-hp-6200:~/pj/ruby_exercise/rspec_square$ gem install rspec


To run all the tests in spec folder
ylee@ylee-hp-6200:~/pj/ruby_exercise/rspec_square$ rspec spec
